About Windows QHOSTS Trojan Cleanup Tool
Restrictions

All Windows users infected by the QHOSTS trojan.

Product
Information

The QHOSTS Trojan Cleanup Tool (vendor: Brown University) repairs the damage done by the QHOSTS trojan.

NOTE: Before running this tool infected users should run LiveUpdate on Symantec AntiVirus and scan their computer.

Requirements
Operating System: Windows 9X/2000/XP
Release date: 2003.10.03

Download and
Installation
  1. Download the QHOSTS Trojan Cleanup Tool by clicking on the link at the bottom of this page.
  2. In the "File Download" window click "Save".
  3. In the "Save As" window accept the default download location and click "Save".
  4. When the download is complete, navigate to your computer's download location and double-click on "clean_qhosts.exe".
  5. Accept the default directory, click "Extract".
  6. Click "OK".
  7. You will be notified when the cleanup is complete.
